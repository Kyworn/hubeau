Mentions Légales

Cadre de Développement

Ce site est un projet de développement personnel et académique, réalisé dans un cadre éducatif et de recherche. Il n'a pas de vocation commerciale et vise à démontrer l'utilisation de données publiques.

Données et Sources

Origine des Données

Les données de qualité de l'eau sont fournies par l'API Hub'Eau, qui fait partie du Système d'Information sur l'Eau (SIE) géré par le Ministère de la Transition Écologique.

Licence des Données

Les données sont disponibles sous Licence Ouverte etalab 2.0. Elles peuvent être librement réutilisées, y compris à des fins commerciales, avec l'obligation de citer la source.

Technologies et Hébergement

Technologies utilisées :

  • Frontend : HTML5, CSS3, JavaScript (ES6+)
  • Framework : Bootstrap
  • Backend : Python
  • API : Hub'Eau

Hébergement :

  • Serveur : Contabo
  • Localisation : Allemagne (IP : 158.220.112.181)
  • Type : Serveur virtuel privé (VPS)

Cookies et Traceurs

Ce site utilise uniquement le stockage local du navigateur (localStorage) pour :

  • Mémoriser le choix du mode sombre/clair

Le localStorage :

  • Ne collecte aucune donnée personnelle
  • Est stocké uniquement sur votre navigateur
  • Peut être effacé à tout moment via les paramètres de votre navigateur

Aucun cookie de tracking ou de collecte de données personnelles n'est utilisé.

Propriété Intellectuelle

Le code source, le design et les éléments graphiques de ce site sont la propriété exclusive de son développeur. Toute reproduction, distribution, modification ou réutilisation sans autorisation explicite est interdite.

Les données provenant de Hub'Eau sont réutilisables selon les termes de la Licence Ouverte Etalab 2.0.

Responsabilité

Ce site est un outil d'information et de visualisation. Les données sont fournies "telles quelles", sans garantie d'exhaustivité, d'exactitude ou de mise à jour.

L'utilisateur est invité à consulter les autorités locales ou les rapports officiels pour toute décision importante relative à la qualité de l'eau.

Protection des Données

Conformément à la loi Informatique et Libertés et au RGPD :

  • Aucune donnée personnelle n'est collectée
  • Aucun profilage n'est réalisé
  • Aucune donnée n'est transmise à des tiers

Clause de Non-Responsabilité

Le développeur de ce site décline toute responsabilité quant à l'utilisation des informations présentées et ne pourra être tenu responsable de tout dommage direct ou indirect résultant de l'utilisation de ces données.

Utilisation des Données Géographiques

Notre site utilise des codes postaux pour localiser et afficher des informations sur la qualité de l'eau :

  • Seuls les codes postaux sont collectés et utilisés
  • Aucune donnée personnelle nominative n'est associée à ces codes
  • Les données sont uniquement utilisées pour rechercher des informations sur la qualité de l'eau

Les codes postaux sont traités de manière anonyme et agrégée, sans possibilité d'identification individuelle.

Évolution et Maintenance du Service

Notre site est un projet en constante évolution :

  • Mises à jour régulières des données de qualité de l'eau
  • Améliorations continues de l'interface et des fonctionnalités
  • Adaptation aux évolutions technologiques et réglementaires

Nous nous réservons le droit de :

  • Modifier, suspendre ou interrompre tout ou partie du service
  • Mettre à jour les fonctionnalités et le design
  • Changer les sources de données ou les API utilisées

Accessibilité

Nous nous engageons à rendre notre site web accessible à tous :

  • Interface conçue pour être lisible sur différents supports (ordinateur, tablette, mobile)
  • Contraste des couleurs optimisé pour une meilleure lisibilité
  • Mode sombre disponible pour réduire la fatigue visuelle
  • Structure HTML sémantique facilitant la navigation

Si vous rencontrez des difficultés d'accessibilité, n'hésitez pas à nous le signaler.
